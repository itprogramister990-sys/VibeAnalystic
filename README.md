## 🌌 Overview

**VibeAI Analytics** is built for AI startups, developers, and data scientists who need absolute visibility into their machine learning models and infrastructure. 

Far beyond a standard admin template, VibeAI provides an immersive, data-dense **Premium Dark / Cyberpunk** interface heavily utilizing custom SVG visualizations, smooth micro-animations, and a responsive glassmorphic design system.

### ✨ Key Features

- **🎛️ Neural Network Activity**: Custom-built interactive SVG charts visualizing inferences per second with neon inflection points and animated scanlines.
- **⚡ Real-Time Mini-Widgets**: Track live API latency, token consumption, AI accuracy bounds, and estimated monthly costs.
- **💻 Live Event Terminal**: A built-in system log emulator to monitor AI model connections, database synchronizations, and server warnings.
- **🎨 Premium UI/UX**: Built entirely on Tailwind CSS with a strict dark mode palette (`slate-950`), custom dot-grid backgrounds, and complex glow/blur effects.
- **🚀 Ultra-Fast Architecture**: Leveraging Next.js 14 App Router, Server Components, and optimized rendering.

## 🚀 Getting Started

### Prerequisites

Ensure you have **Node.js 18+** installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/itprogramister990-sys/VibeAnalystic.git
   cd VibeAnalystic
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Explore the Dashboard:**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the live analytics interface.
